# Domain Vocabulary Reference

A curated vocabulary bank organized by industry and role. Use these terms to signal insider fluency and match the reader's professional language context.

---

## Tech / SaaS

Terms used by builders, engineers, and product-oriented founders in software companies.

| Term | Context Note |
|------|-------------|
| ship | Preferred over "launch" in builder culture; implies bias to action |
| iterate | Signals agile mindset; implies feedback-driven improvement |
| scale | Can mean technical or business scaling; context distinguishes |
| deploy | Technical: push code to production; signals hands-on builders |
| pipeline | Data or CI/CD; insider shorthand for systematic workflow |
| sprint | Two-week work cycle; Scrum vocabulary widely adopted beyond engineering |
| backlog | Prioritized list of future work; universal in modern product teams |
| DX | Developer experience; shorthand signals awareness of tooling quality |
| latency | Milliseconds matter; shows awareness of performance trade-offs |
| throughput | Volume over time; pairs with latency in systems thinking |
| microservices | Architectural pattern; signals distributed systems experience |
| API-first | Design philosophy; signals platform and integration thinking |
| observability | Beyond monitoring; tracing, metrics, and logs together |
| zero downtime | Deployment quality bar; signals production maturity |
| feature flag | Controlled rollout mechanism; signals safe deployment culture |
| mono-repo | Single repository for multiple packages; signals infra awareness |
| SLA | Service level agreement; reliability commitment vocabulary |
| on-call | Operational responsibility rotation; signals production ownership |

## AI / ML

Terms used by researchers, engineers, and practitioners in machine learning and AI.

| Term | Context Note |
|------|-------------|
| inference | Running a trained model to get predictions; vs. training |
| fine-tuning | Adapting a pre-trained model to a specific task |
| embeddings | Dense vector representations of text, images, or audio |
| RAG | Retrieval-augmented generation; grounds LLMs in external data |
| prompt engineering | Crafting inputs to elicit desired model outputs |
| hallucination | When a model generates confident but factually wrong output |
| foundation model | Large pre-trained model used as base for fine-tuning or prompting |
| agentic | AI systems that plan and take actions autonomously over time |
| context window | Maximum token input a model can process at once |
| token | Base unit of text for LLMs; ~0.75 words on average |
| grounding | Connecting model outputs to verifiable real-world facts |
| alignment | Ensuring model behavior matches intended human values |
| RLHF | Reinforcement learning from human feedback; training technique |
| synthetic data | Machine-generated training data; addresses data scarcity |
| multimodal | Models that process text, images, audio simultaneously |
| chain-of-thought | Prompting technique eliciting step-by-step reasoning |
| vector database | Storage optimized for similarity search on embeddings |
| quantization | Reducing model precision to lower compute cost |

## Product Management

Terms used by PMs, product leaders, and cross-functional collaborators.

| Term | Context Note |
|------|-------------|
| north star metric | Single KPI that best captures product value creation |
| OKR | Objectives and Key Results; goal-setting framework |
| discovery | Research phase before building; validates problem before solution |
| user story | "As a [user] I want [X] so that [Y]"; structured requirement format |
| RICE | Reach, Impact, Confidence, Effort; prioritization scoring |
| sprint | Time-boxed iteration; planning and execution unit |
| roadmap | Forward-looking plan of features and initiatives |
| feature flag | Toggle to release features to subsets of users |
| A/B test | Controlled experiment comparing two variants |
| jobs-to-be-done | Framework: what outcome does the user hire this for? |
| MVP | Minimum viable product; smallest shippable version |
| PMF | Product-market fit; when product strongly satisfies a market |
| retention curve | Graph showing how many users return over time |
| activation | Moment when a new user first experiences core product value |
| churn | Rate at which users stop using or paying for a product |
| DAU/MAU ratio | Engagement metric; daily to monthly active user ratio |
| user journey | End-to-end path a user takes through a product experience |
| stakeholder alignment | Ensuring all decision-makers share understanding and buy-in |

## Engineering Management

Terms used by engineering managers, CTOs, and technical leaders.

| Term | Context Note |
|------|-------------|
| velocity | Story points or tasks completed per sprint |
| retro | Retrospective meeting; team reflection on process |
| 1:1 | Weekly manager-report check-in; relationship and growth conversation |
| IC track | Individual contributor career path; vs. management track |
| tech debt | Accumulated shortcuts that slow future development |
| code review | Peer review of code before merging; quality and knowledge sharing |
| on-call | Rotation for responding to production incidents |
| incident response | Structured process for detecting and resolving production outages |
| blameless postmortem | Incident review focused on systems, not people |
| hiring funnel | Stages from sourcing to offer in recruiting |
| span of control | Number of direct reports a manager oversees |
| eng culture | Shared values and practices within an engineering organization |
| platform team | Team building internal tooling and infrastructure for other teams |
| staffing ratio | Balance of senior to junior engineers on a team |
| growth model | Documented competency levels for IC and manager tracks |
| runbook | Step-by-step operational procedure for common incidents |
| headcount | Number of allocated engineering positions |
| psychological safety | Team climate where members feel safe to speak up and take risks |

## Marketing

Terms used by marketers, growth teams, and demand generation professionals.

| Term | Context Note |
|------|-------------|
| funnel | Stages from awareness to conversion; top/mid/bottom |
| conversion | Desired action taken; click, signup, purchase |
| CAC | Customer acquisition cost; total spend per new customer |
| LTV | Lifetime value; total revenue expected from a customer |
| attribution | Crediting which touchpoints drove a conversion |
| persona | Semi-fictional ideal customer profile used for targeting |
| content calendar | Planned schedule of content topics and publish dates |
| engagement rate | Interactions divided by reach or impressions |
| MQL | Marketing qualified lead; passed threshold to hand to sales |
| SQL | Sales qualified lead; sales team confirmed interest |
| ICP | Ideal customer profile; characteristics of best-fit customers |
| GTM | Go-to-market; strategy for launching and selling a product |
| demand generation | Activities that create awareness and interest in a product |
| inbound | Leads that find you through content and search |
| outbound | Proactive outreach to potential customers |
| TOFU/MOFU/BOFU | Top/middle/bottom of funnel; content targeting by stage |
| share of voice | Brand's presence relative to competitors in a market |
| retargeting | Re-engaging users who previously visited but didn't convert |

## Finance / Business

Terms used by founders, CFOs, investors, and operators focused on business metrics.

| Term | Context Note |
|------|-------------|
| runway | Months of operating capital remaining at current burn |
| burn rate | Monthly cash outflow exceeding revenue |
| MRR | Monthly recurring revenue; SaaS subscription metric |
| ARR | Annual recurring revenue; MRR × 12 |
| churn | Percentage of revenue or customers lost per period |
| unit economics | Per-customer or per-transaction profit and loss |
| TAM | Total addressable market; maximum revenue opportunity |
| Series A/B/C | Venture capital funding rounds by stage |
| cap table | Capitalization table; ownership breakdown of equity holders |
| dilution | Reduction in ownership percentage from new share issuance |
| EBITDA | Earnings before interest, taxes, depreciation, amortization |
| gross margin | Revenue minus cost of goods sold, as a percentage |
| bridge round | Interim financing between major funding rounds |
| down round | Funding at a lower valuation than the previous round |
| liquidation preference | Order in which investors are paid in an exit event |
| path to profitability | Plan and timeline to reach positive operating cash flow |
| blended CAC | Weighted average acquisition cost across all channels |
| rule of 40 | SaaS health metric: growth rate + profit margin ≥ 40% |

## Leadership

Terms used by executives, senior leaders, and people in organizational influence roles.

| Term | Context Note |
|------|-------------|
| alignment | Shared understanding and commitment across stakeholders |
| stakeholder | Anyone with interest or influence in a decision or outcome |
| north star | The guiding metric or vision that orients all decisions |
| OKR | Objectives and Key Results; cascading goal framework |
| cross-functional | Working across multiple teams or disciplines |
| executive presence | Commanding attention and trust in leadership settings |
| organizational design | How teams, roles, and reporting lines are structured |
| change management | Process of helping people adopt new ways of working |
| operating cadence | Rhythm of recurring meetings, reviews, and decisions |
| delegation | Assigning tasks with appropriate authority and accountability |
| span of control | Number of direct reports a leader effectively manages |
| strategic clarity | Clearly communicating direction so teams can make decisions |
| culture carrier | Person who actively embodies and spreads organizational values |
| two-pizza team | Amazon heuristic: team small enough to feed with two pizzas |
| decision rights | Documented clarity on who can decide what |
| narrative alignment | Ensuring all leaders tell a consistent story about direction |
| board communication | Style and cadence of updates to board of directors |
| succession planning | Developing leaders ready to fill critical roles |
